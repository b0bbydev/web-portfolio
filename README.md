# Web Portfolio
This is the source code for my online web portfolio. It is mostly an on-going project as I like to add new features here, and practice my web development skills.
During the development of this project I've learned how to set up a hosting service with a domain name, deploy a database online, and manage misc. information within the website.

## Technologies Used
* PHP - used for backend development.
* JavaScript - used for some formatting on banner, header and background images. Also included jQuery as well as an Ajax plugin called ```TableEdit``` for convenience of database CRUD functionality.
* HTML5/CSS3 - for website structure and design.
* MySQL Workbench/phpMyAdmin - used for database management system.
